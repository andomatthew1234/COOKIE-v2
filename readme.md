## 🖥 Compatibility
* 🍎 **macOS** 
* 🖥 **Windows** *Windows 10 or later recomended*
* 🐧 **Linux/other OSes** *not guaranteed usage - check your OS supports HTML.*
* 🕸 **Internet browser (eg. Chrome, Edge, Firefox)** *required for running HTML file*

## 🚀 Let's get started
1. **Download** by selecting the green code button > Download ZIP. 
2. **Extract** the files. I know you think it doesn't matter, but for Cookie Clicker to work properly, right click on the .zip file you downloaded in your File Explorer, and select 'Extract all'. Choose a location and press extract.
2. **Double click** on the index.html file in the folder that just opened and Cookie clicker will open in your browser!
